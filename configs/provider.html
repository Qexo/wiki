自动部署 ​

自动部署命令, 将在每次源码修改后执行, 若留空则默认不执行

bash
hexo clean && hexo g && hexo d
1
bash
hugo
1

如果提示 /bin/sh: 1: xxx : not found

请尝试在每一条命令前加上 /bin/bash -c

bash
/bin/bash -c "hexo clean" && /bin/bash -c "hexo g"
