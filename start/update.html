如何更新 ​

一般情况下, 推荐使用 Qexo 的一键更新功能

一键更新 ​

Vercel 部署 ​

  • 确保你当前的 Qexo 版本在 1.5.0 以上或 Dev 分支
  • 打开设置, 点击“一键更新”, 选择更新通道(dev->开发版 master->稳定版)

本地部署 ​

  • 确保你当前的 Qexo 版本在 2.0.0 以上或 Dev 分支
  • 打开设置, 点击“一键更新”, 选择更新通道(dev->开发版 master->稳定版)
  • 你可能需要等待十秒钟的程序自动重启

Tips: 本地部署一键更新不会删除 db/ 和 configs.py

手动更新 ​

Vercel 部署 ​

  • 创建 Pull Request 并拉取 Qexo 主分支源码

本地部署 ​

  • 参考 本地部署
